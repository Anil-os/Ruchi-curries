# Ruchi Curries - Image Placeholder

This file indicates where you need to place your images.

## Required Images:

Place the following images in the root directory (same folder as index.html):

1. **ruchi-removebg-preview-1.png** - Logo image
2. **biriyani-removebg-preview-2.png** - Biryani bottom image
3. **biriyani-removebg-preview-3.png** - Biryani background image
4. **kebab-removebg-preview-1.png** - Kebab bottom image
5. **kebab-removebg-preview-2.png** - Kebab background image
6. **mutton-removebg-preview-1.png** - Mutton bottom image
7. **mutton-removebg-preview-2.png** - Mutton background image
8. **prawns-removebg-preview-1.png** - Prawns bottom image
9. **prawns-removebg-preview-2.png** - Prawns background image

## Export from Figma:

1. Open your Figma design
2. Select each image layer
3. Export as PNG with transparent background (remove background)
4. Save with the exact names listed above
5. Place all images in the root folder

## To Test:

After placing the images, open `index.html` in your browser or run a local server:

```bash
python -m http.server 8000
```

Then navigate to http://localhost:8000
